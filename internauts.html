internauts

A Ruby on Rails app sports pick-up game finder. Users can create accounts to start or join desired pick-up games in their area. Making pickup games easier to get started and played.

User Content

Users have all the power. Utilizing the rails framework, users generate the content with forms that output all the data into a database to dynamically design pages and input create google maps

Included Fun

Google maps API allow viewers to easily locate a pick-up game. Users dynamicly update data on the page and the database by joining a game or sharing a comment to connect to people in their area.
