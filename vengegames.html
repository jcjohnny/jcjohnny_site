Vengegames

One of the stats that rarely get tracked for the NBA are revenge games. Revenge games are categorized when a player is playing against their former team. In this application, I track all the revenge games for the current day, and display the average revenge stats compared to current season averages to check if a player performs better during revenge games.

All Angular

This applications built completely with Angularjs to test the full capabilities of the framework. Just the A in the mean stack coupled with a a few libraries to create a functioning app that operates without a need for a server file in the application.

It's about the stats

Built mainly for fantasy basketball sports fans in mind. Vengegames proves to be a helpful resource for fans that are looking for players that may have a higher performance rating during revenge games as opposed to a regular season game. This helps the user decide if a player is worth selecting temporarily as a part of their fantasy league.

API level above 1000

With the NBA api being highly robust, it took some some time to figure it out. Depending on the amount of games being played for that day, the application can make hundreds of api calls during one session to create the data: discovering, compiling and creating information all on the client side.
