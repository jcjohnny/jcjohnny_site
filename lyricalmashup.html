Lyrical Mashup

A Node.js web application that creates music! By inputting a few words - lines of lyrics will be generated with those words as its parameters and it can be played, with background music and voices.

Total Randomness

Each word inputted by the user will input a single line of lyric from a random song and artist. Each words grabs a random line of lyric from a list generated from the MusixMatch api creating a mashed up song that can be sung by several different voices and background music. Completely at random.

Create or Find music

Combines the spotify API so users can learn about where that line of music is generated from as well as play the song on spotify on their machine.
